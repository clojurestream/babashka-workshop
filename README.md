# Babashka Workshop

## System Requirements

- [Babashka][babashka]

To verify things are set up properly, you can run this:
```
bb --version

babashka v<version-installed>
```

## Run

```
git clone https://github.com/clojurestream/babashka-workshop

cd babashka-workshop/exercies/00-run
```
